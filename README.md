# Sleeping Gods Companion

A campaign tracker web app for the **Sleeping Gods** board game by Red Raven Games.

## Features

- **Player & Character Management** — Assign the 8 crew members across players, track command tokens and cards in hand
- **Ship Dashboard** — Track ship location, damage across 6 sections, resources, and last ship action
- **Experience Points** — Earn and spend XP with an intuitive counter
- **Atlas Location Tracker** — Visit locations, look up atlas page numbers, name locations, and track keywords
- **Keyword System** — Track keywords at locations (locked/unlocked) and keywords acquired by players
- **Quest & Totem Journal** — Track active quests and collected totems
- **Character Status** — Track damage, fatigue, card slots, upgrades, weapons, and status tokens per character
- **Session Save/Resume** — Auto-saves to localStorage; full summary screen between sessions
- **PWA Support** — Install as a homescreen app on phone or tablet with offline support

## Deployment to GitHub Pages

1. Create a new GitHub repository
2. Push all files in this folder to the repository's `main` branch
3. Go to **Settings → Pages** and set the source to `main` branch, root folder
4. Your app will be live at `https://<username>.github.io/<repo-name>/`

## Usage

- **Phone**: Designed for portrait orientation at 420px width
- **Tablet**: Responsive layout at 1186px+ width
- **Add to homescreen** for the best experience (acts as a standalone app)

## Tech Stack

Pure HTML, CSS, and vanilla JavaScript — no build step required. Persistence via `localStorage`.

## Theme

1920s nautical/vintage aesthetic inspired by the game's setting aboard the steamship Manticore in 1929, featuring Playfair Display and Crimson Text fonts with a navy/brass/parchment color palette.
